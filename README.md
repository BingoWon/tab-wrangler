# Tab Wrangler

Tab Wrangler 是一个我为自己日常使用写的小型浏览器插件。

这不是一个准备推广的产品，也没有流量或热度预期。作为一名开发者管理者，我只是希望把自己做过、自己觉得有用的小工具开放出来，方便别人看到时能大概了解它做了什么。

## 功能

- 在同一个浏览器窗口里，只保留同一个 URL 的一个标签页。
- 打开重复标签页时，自动切回已经存在的标签页，并关闭新打开的重复标签页。
- 切走后自动关闭没有使用的空白页或新标签页。
- 关闭当前标签页后，尽量回到上一个使用过的标签页。
- 恢复刚关闭的标签页时，如果浏览器支持，尽量把它移回原来的位置。

## 项目结构

- `manifest.json` 和 `js/background.js` 是共享的 WebExtension 源码。
- `images/` 存放插件图标。
- `Tab Wrangler Safari/` 是 Safari 包装工程，会打包同一份共享插件源码。

## 说明

这个插件刻意保持简单，没有弹窗页、设置页、账号系统、统计分析或远程服务。

Safari 不会让扩展的 service worker 一直常驻运行，所以插件按事件驱动设计，并在被唤醒时恢复需要的状态。
